Never Miss List

First Step - Create An Account

1. Create an account by clicking the Sign Up tab in the top right corner of the page

2. Enter Your information, which includes creating and inputting

  • Username
  • Last Name
  • First Name
  • Email
  • Password
  • Password Confirmation

3. After creating an account, you'll be directed to your list profile

In this page you can create a an event to add to your list by clicking the "Create Event" button

4. Give the event an name, add the date, time, and a description and click on the "Create Event" button

5. To see the description of an event just click the on the rectangle with the event name and it should collapse with the description

6. You can edit and delete the event after creation by clicking either the "Delete" or "Edit" button
